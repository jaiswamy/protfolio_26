Highly Available AWS EKS Cluster Using Terraform

Project Overview

Designed and provisioned a production-ready AWS EKS cluster using modular Terraform. Implemented high availability, security best practices, and automated backup strategies.

Architecture

  • Custom VPC with Public & Private Subnets
  • EKS Control Plane with Managed Node Groups
  • IRSA for secure service-to-AWS access
  • EBS CSI Driver for dynamic volume provisioning
  • AWS ALB for Load Balancing

Security Implementation

  • IAM Roles for Service Accounts (Least Privilege)
  • Namespace Isolation
  • RBAC Policies
  • Automated EBS Snapshot Backup Strategy

Tools & Technologies

AWS, Terraform, Kubernetes, IRSA, EBS CSI, ALB

Impact

  • High Availability Architecture
  • Automated Infrastructure via IaC
  • Secure workload isolation
