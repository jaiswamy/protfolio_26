GitOps-Driven Veterinary Portal Deployment

Client

Karnataka Veterinary University (KVAFSU)

Project Overview

Designed and deployed a production-grade veterinary job application portal using Dockerized 3-tier architecture on AWS. Implemented automated CI/CD pipelines and enforced strict security segmentation for high-load production traffic.

Architecture

  • Custom VPC in ap-south-1
  • Frontend, Backend & Database tier separation
  • Docker & Docker Compose-based deployment
  • Secure subnet design with strict traffic segmentation

CI/CD Implementation

  • GitHub commit triggers Jenkins pipeline
  • Automated WAR build process
  • Docker image creation & deployment automation
  • Version-controlled infrastructure provisioning using Terraform

Security Implementation

  • My-IP-only SSH restriction
  • Frontend accessible only from Backend SG
  • Database accessible only from Backend SG
  • Strict inbound/outbound traffic control

Monitoring & Observability

  • Prometheus for metrics collection
  • Grafana dashboards for real-time monitoring
  • Server health and container metrics tracking

Impact

  • Production-ready deployment
  • Improved deployment consistency
  • Secure architecture for payment workflows
