DevOps-Driven Multi-Tier Microservices Platform

Project Overview

Provisioned a scalable AWS EKS-based microservices architecture using Terraform. Implemented GitOps workflows with ArgoCD and applied fine-grained IAM policies to enhance security and reliability.

Infrastructure Setup

  • Provisioned AWS EKS cluster using Terraform & eksctl
  • Configured Managed Node Groups
  • Automated environment setup using Bash scripts

GitOps Implementation

  • Implemented ArgoCD for continuous deployment
  • Automated Kubernetes manifest synchronization
  • Reduced deployment time by 30%

Security & IAM

  • Constructed fine-grained IAM roles
  • Applied Least Privilege principle
  • Secured service account access

Monitoring & Performance

  • Grafana dashboards for cluster metrics
  • Optimized resource allocation
  • Prevented unnecessary scaling

Impact

  • Improved system reliability
  • Accelerated deployment cycles
  • Enhanced security posture
